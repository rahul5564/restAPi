The Difference between copy by value and copy by reference?

"Copy by value" is like assigning a variable value to another variable,if the value of first variable it doesnt affect the second one,If a value is assigned to b ,And the a value changes it doesnt effect the value of b.

"Copy by reference" its like an array1 value is assinged to another array2,if we change the 1st value it effects the second value.
